# Moloco adapter using IronSource mediation Unity Demo App

A demo app that shows how to load and show ads using Moloco Unity plugin.
Dependency versions:
- Moloco IronSource adapter 1.2.x
- Unity 2021.3.27f1
- IronSource 7.3.1.1
